# 🔁 Fluxo dos Sistemas

Este projeto funciona baseado em um sistema de eventos e respostas.

## 🧠 Como funciona:

1. O usuário executa uma ação (comando)
2. O sistema interpreta essa ação
3. Um evento é disparado
4. As entidades ou sistemas reagem
5. Uma nova ação pode ser gerada

---

## ⚙️ Exemplo prático:

Comando → Criar NPC  
Sistema → Registra entidade  
NPC → Executa comportamento automático  
Resultado → Interação dinâmica no ambiente

---

## 💡 Ideia principal:

Tudo no sistema é conectado por eventos, criando um fluxo contínuo de ações e respostas.
