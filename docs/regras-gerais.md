# 📌 Regras Gerais do Sistema

Este documento define como o sistema deve se comportar.

---

## ⚙️ Princípios principais:

- Tudo funciona baseado em lógica de eventos
- Nenhuma ação acontece sem um comando ou gatilho
- Entidades possuem comportamento definido
- Sistemas devem responder de forma automática

---

## 🧩 Organização:

- Comandos → iniciam ações
- Sistemas → processam lógica
- Entidades → executam comportamentos
- Eventos → conectam tudo

---

## 🚫 Limitações:

- Evitar ações sem controle lógico
- Evitar loops sem finalização
- Toda ação deve ter resposta definida
